# AOP(Aspect Oriented Programming)

## AOP 적용

1. 회원가입, 회원 조회 등 핵심 관심사항과 시간을 측정하는 공통 관심 사항을 분리한다.
2. 시간을 측정하는 로직을 별도의 공통 로직으로 만들었다.
3. 핵심 관심 사항을 깔끔하게 유지할 수 있다.
4. 원하는 적용 대상을 선택할 수 있다.
